# Fitness Tracker Application

## Overview

A C# Windows Forms application developed for fitness activity management. The system allows users to register, log in, record activities and view fitness history.

## Features

- User registration
- User login authentication
- Activity tracking
- Fitness history management
- Data processing and reporting

## Technologies Used

- C#
- .NET Windows Forms
- Object-Oriented Programming
- Visual Studio

## Skills Demonstrated

- Software Development
- Object-Oriented Programming
- User Authentication
- Windows Forms Development
- Testing and Debugging
